Variables in JavaScript

Variable is named storage location to store,process,manipulate,retrive the values during the execution of of program

To create variable names in JS we use identifiers

Types of variables

  1. Local variable
  2. Global variabe

Local variable:-

A variable which declared inside a block of code. Local variable is declared by using "Let" keyword

Global variable:-

A variable which declared outside of block of code.Global variable is declared by using "var" keyword.
