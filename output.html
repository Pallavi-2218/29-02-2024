Output Statements In JavaScrit

There are 4 types of output statements in JavaScrit.
They are

  1. document.write()
  2. innerHTML
  3. console.log
  4. alert

document.write()

We can print HTML elements directly on the document(active webpage)

innerHTML

Used to pass the HTMl data to already existing HTML element

Used to pass the HTMl data to already existing HTML element

console.log

Used to print content in console of the browser

alert

Used to print content in dialogue box
